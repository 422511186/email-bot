# 项目概述

## 项目简介

Email Bot 是一个使用 Go 语言编写的**终端邮件转发机器人**，旨在帮助用户将多个邮箱账户的邮件自动转发到指定目标地址。

### 核心能力

```
源邮箱（IMAP） → Email Bot → 目标邮箱（SMTP）
     ↓
  定时轮询
     ↓
  增量获取新邮件
     ↓
  转发到目标地址
```

## 核心功能

| 功能 | 说明 |
|------|------|
| **增量拉取** | 仅获取新邮件，不重复处理历史邮件 |
| **首次运行保护** | 首次启动时自动跳过历史邮件 |
| **一对多转发** | 一个源邮箱可转发到多个目标地址 |
| **多对多配置** | 支持多个源邮箱，每个有独立目标列表 |
| **多协议支持** | IMAP (993/143) + SMTP (587/465) |
| **TLS 加密** | 支持 STARTTLS 和隐式 TLS |
| **实时 TUI** | 终端界面实时显示状态、绑定信息（IMAP/文件夹/目标地址）和日志 |
| **跨平台** | Windows / Linux / macOS |
| **状态持久化** | 重启后从上次位置继续 |

## 技术栈

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| 语言 | Go 1.21+ | 高并发、内置 TLS、网络库优秀 |
| IMAP 客户端 | go-imap v1 | 活跃维护、API 清晰 |
| TUI 框架 | Bubbletea v1 | 优雅的终端 UI 库、Elm 架构 |
| 样式库 | Lipgloss | 与 Bubbletea 配套、美观 |
| 配置格式 | YAML | 易于阅读和维护 |
| YAML 解析 | gopkg.in/yaml.v3 | 稳定、广泛使用 |

```

```

## 应用场景

1. **个人邮件聚合**：将多个邮箱的邮件汇聚到一个收件箱
2. **邮件备份归档**：自动将重要邮件转发到归档邮箱
3. **多平台通知**：将工作邮件转发到个人邮箱或即时通讯
4. **邮件过滤转发**：配合邮件规则实现自动分类转发

## 设计原则

1. **简单可靠**：最小依赖，最少配置
2. **幂等处理**：重复运行不会导致重复转发
3. **优雅退出**：支持平滑关闭，不丢邮件
4. **用户友好**：TUI 界面，操作直观
